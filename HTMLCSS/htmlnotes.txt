HTML (Hyper Text Markup Language)
It is not a programming Language

The purpose of HTML is to display information and content on a webpage
the HTML file extension tells the browser to interpret and parse the 
characters to create a webpage

Elements - core building block of an HTML page
They define the strucure
They can be nested. Generally bad practice to overlap or not close

Browsers tend to be very fault tolerant

<h1>value of the element</h1>

Attributes - adjective to an element 
key value relationship

CSS (Cascading Style Sheets)
To make our web pages look pretty and easy and approachable for humans to use

The cascade algorithm determines what the final look of an html element 
the most specific locator wins (inline always wins)

Bootstrap - CSS library developed by Twitter
